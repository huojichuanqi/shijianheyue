事件合约自动下单脚本

该脚本某位群友开发，我用了一段时间，很不错，根据2个多月的实盘，长期胜率应该在58-60%之间（策略设计好之后的回测有60%+，但交易所点杀会造成影响），考虑到bn跟hibt都是交易所跟用户对赌，不存在流动性不足问题，理论上资金容量巨大，但正因为交易所对赌，所以看格局了，hibt 赔率高但是封号解封麻烦，bn系统性风险小，即使封号也只是封禁事件合约权限。

软件已经技术层面做了风控（不是简单风控），但是不保证不封号，如果本金大建议找客服做订单分流（多账户多ip）
